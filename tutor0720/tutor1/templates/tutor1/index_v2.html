Spine segmentation mission with Unet

In order to exceed the original baseline ( 0.71 ), I add additional layers to increase the ability to extract feature. Besides more layers, I also add a post-process based on image processing, which can roughly remove the noice ( false recognized area ). However, this process usually can't led to a higher average Dice score because some dispersed positive area might be incorrectly recognized as noice, which makes it be removed and thus results in a lower score.
